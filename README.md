# password-generator
A completely self-built, reliable password generator

This is my own completely random password generator created entirely on my own.

The design came from the Scrimba Frontend Developer Course, with my own modifications and additions. 

These include the ability to modify the parameters of the password, such as the length and type of characters used. 

Check out the code live: https://rishipasswords.netlify.app

Please feel free to test it and let me know any feedback you have.

Enjoy!
